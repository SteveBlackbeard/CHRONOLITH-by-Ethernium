# Security Policy



## Supported Versions



Currently, the latest release on the `Ethernium` branch is supported for security updates.



| Version | Supported          |

| ------- | ------------------ |

| v1.0.x  | :white_check_mark: |

| < 1.0   | :x:                |



## Reporting a Vulnerability



We take the security of this project seriously. If you find a security vulnerability, please do NOT open a public issue. Instead, report it privately to ensure the safety of all users.



To report a vulnerability:

- Send an email to **steveblackbeard@users.noreply.github.com** (or reach out via X: **@Steveblackbeard**)

- Provide a detailed description of the vulnerability.

- Include steps to reproduce the issue.



We will acknowledge your report and work on a fix as soon as possible. Once the fix is ready, we will release an update and credit you for the discovery.
