# CONTINUITY LITE ⚡🌱

**The Zero-Overhead AI Continuity Engine.**

This is a stripped-down, high-speed implementation of the CONTINUITY LEGACY framework, designed for personal projects, rapid prototyping, and solo AI-pair-programming.

## 🚀 Quick Start (60 Seconds)

1.  **Initialize**: Create three files in your root:
    - `PROJECT_CONTEXT.md`: The "Bible" of your project (Vision, Rules, Stack).
    - `STATE.json`: The "Pulse" (Current status, next steps).
    - `LIVE_HANDOFF.md`: The "Relay" (Exact next action for the AI).

2.  **Run Sync**: 
    ```bash
    python run_continuity_lite.py
    ```
    This updates your `STATE.json` with Git metadata and validates that your core "Truth Documents" exist.

## 🧠 Why Lite?

- **Zero Dependencies**: Pure Python 3. Standard library only.
- **Git Native**: Auto-detects branch and commit hashes.
- **Low Ceremony**: No `registry/`, no `outputs/`, no `hooks/`. Just three files and one script.
- **AI-First**: Designed to be the first thing an AI reads to understand the mission.

## 🛠️ Evolution

When your project grows and needs industrial-grade governance, security scanning, and visual dashboards, simply migrate to the **[CONTINUITY LEGACY FULL](..)** version.
