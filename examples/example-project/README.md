## 🚀 Cómo usar este ejemplo

1. **Entrar al directorio**:
   ```bash
   cd example-project
   ```

2. **Ejecutar el Guardián de ADN**: Escaneará la configuración, analizará el núcleo de memoria y producirá el resumen cognitivo:
   ```bash
   chronolith-lite
   ```

3. **Demostración de Paridad**: 
   Abre `STATE.json` o `.chronolith/LIVE_HANDOFF.md`, modifica algo (ej. cambia una versión o una decisión) y vuelve a ejecutar `chronolith-lite`. Observa cómo el Merkle Tree del ADN se desplaza para capturar la nueva iteración.
